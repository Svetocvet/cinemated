API v1

Hello, thank you for using our service. Please note that in order to use our api
you should have the role of administrator. Our api has such features:

Getting users:

Method: GET

Url: /api/v1/users

Params:

page (not required) Get users by page
search (not required) Get users by title or part of title

Return: users(with pagination)

Adding users:

Method: POST

Url: /api/v1/users

Params:

username Unique string
password Very hard string
fullname Some string

Return: created user

Getting one user by id:

Method: GET

Url: /api/v1/users/:id

Params: No params

Return: user

Updating one user by id:

Method: PUT

Url: /api/v1/users/:id

Params:

role 0(User) or 1(Admin)
fullname Some string

Return: No content

Deleting one user by id:

Method: DELETE

Url: /api/v1/users/:id

Params: No params

Return: No content

Films

Getting films

Method: GET

Url: /api/v1/films

Params:

page (not required) Get films by page
search (not required) Get films by title or part of title

Return: films(with pagination)

Inserting film

Method: POST

Url: /api/v1/films

Params:

object Film object
title Some string
link url to some film
poster url to some poster

Return: created film

Getting one film by current id

Method: GET

Url: /api/v1/films/:id

Params: No params

Updating film by current id

Method: PUT

Url: /api/v1/films/:id

Params:

length positive number
quality positive number
title Some string

Return: No content

Deleting film by current id

Method: DELETE

Url: /api/v1/films/:id

Params: No params

Return: No content
